%Abstrakt (cesky, slovensky ci anglicky) / Abstract (in czech, slovak or english)
\abstract[cs]{Práce popisuje implementaci jednoduché 3D hry pomocí OpenGL. Věnuje se návrhu aplikace a konkrétní implementaci, která je výstupem této práce. V textu práce je postupně popsán způsob použití knihoven \emph{Assimp}, \emph{Bullet} a \emph{IrrKlang}.  \emph{Assimp} pro načítání souborů formátu \emph{dae} a \emph{obj}. \emph{Bullet} pro zachytávání kolizí a fyziku scény. A \emph{IrrKlang} jako nástroj pro 3D ozvučení. Dále se práce podrobně věnuje osvětlení, stínům a implementaci \emph{Skeleton animation}. Část práce je také věnována použité implementaci \emph{skyboxu} a střídání dne a noci. V poslední kapitole jsou poté popsány výkonnostní testy výsledné implementace.	
}
\abstract[en]{This Bachelor's thesis describe implementation of simple 3D game by OpenGL. The attention is paid to draft of this game and specific implementation, which is output of this work. In text of thesis is described usage of libraries \emph{Assimp}, \emph{Bullet} and \emph{IrrKlang}. \emph{Assimp} for loading  an \emph{.obj} and \emph{.dae} model file. \emph{Bullet} for detection of collision and physics of scene. And \emph{IrrKlang} for sounds. Furthmore thesis  deals with lighting, shadows and implementation of \emph{Skeleton animation}. Part of thesis is dedicated to implementation of \emph{skybox} and changing day and night. In last chapter are analyzed test of game.
}
